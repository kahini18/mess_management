Mess Management System:

A Python-based mess management system for managing members, meal entries, expenses, and monthly final reports.

## Project Motivation

This project is inspired by my friend, Auri's manually maintained mess management system. He helped me understand how mess records are managed in real life, including member details, daily meals, expenses, deposits, and final monthly calculations. Based on that understanding, I built this application to reduce manual work and make mess management more organized and reliable. 

## Features
- Add, update, delete members
- Date-wise lunch and dinner entry
- Expense tracking
- Monthly final report generation
- CSV report export
- Search records
- Clean GUI using streamlit

## Technologies Used
- Python
- SQLite
- streamlit
- pandas
